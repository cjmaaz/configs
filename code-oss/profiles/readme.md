## Quick Start

### Import a Profile

1. Open Command Palette: `Cmd/Ctrl + Shift + P`
2. Type: "Profiles: Import Profile"
3. Select "Select File" â†’ Browse to `.code-profile` file
4. Click "Create Profile"
5. Name the profile (or keep default)

### Switch Profiles

1. Command Palette: `Cmd/Ctrl + Shift + P`
2. Type: "Profiles: Switch Profile"
3. Select desired profile

Or use the profile switcher in the bottom-right corner of IDE.
